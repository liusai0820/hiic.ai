# HIIC.gwy.life 应用矩阵首页

这是HIIC.gwy.life应用矩阵的官方首页，展示了我们的AI应用产品线，包括：

- 智能项目管理平台 (PMP.AI)
- 公文笔杆子
- 智绘链图
- 智能改稿助手

## 部署

本项目使用Vercel进行部署，访问地址：https://hiic.gwy.life

## 本地开发

要在本地运行此项目，请执行以下命令：

```bash
# 使用任意HTTP服务器运行
python -m http.server
# 或
npx serve
```

然后在浏览器中访问 http://localhost:8000
